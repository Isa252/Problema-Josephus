TÍTULO DO PROJETO: Problema do Josephus
OBJETIVO DO PROJETO:  O objetivo deste projeto é simular o clássico
problema matemático conhecido como o Problema do Josephus, onde um grupo
de indivíduos está em uma formação circular e são eliminados um a um em um 
padrão específico, até que apenas um sobreviva.
VERSÃO ou DATA: 20/06/2024
COMO INICIAR O PROJETO: Para iniciar o projeto, compile a classe "Teste". 
Isso abrirá uma interface com 5 botões, 3 campos de texto e uma tabela dos 
indivíduos.
AUTORES: Isabela Groke Gomes
INSTRUÇÕES PARA O USUÁRIO:
O programa começa com um número fixo de indivíduos, que pode ser ajustado pressionando o botão "Configurar".
Certifique-se de que nenhum dos números inseridos seja negativo, 
pois isso pode causar erros no funcionamento do programa.
